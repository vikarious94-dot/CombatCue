# CombatCue

CombatCue is a lightweight World of Warcraft Retail addon that displays a customizable visual alert when your character enters or leaves combat.

By default, it shows:

- `++combat++` when entering combat
- `--combat--` when leaving combat

## Features

- Custom enter-combat and leave-combat messages
- Custom text color and alpha
- Adjustable font size
- Adjustable screen position with sliders or direct input
- Draggable preview text
- Optional alert animations: fade, scale, or flash
- English and French interface

## Installation

1. Download or clone this repository.
2. Copy the `CombatCue` folder into:

```text
World of Warcraft/_retail_/Interface/AddOns/
```

3. Make sure the final path looks like:

```text
World of Warcraft/_retail_/Interface/AddOns/CombatCue/CombatCue.toc
```

4. Restart World of Warcraft or reload your UI.
5. Enable `CombatCue` from the AddOns menu.

## Usage

Open the configuration window with:

```text
/combatcue
/cc
/cs
```

From the configuration window, you can edit messages, colors, font size, animation settings, and alert position.
